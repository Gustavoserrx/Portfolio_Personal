Proyecto Web TeatroGest

Proyecto de final de grado donde junto a un equipo utilizamos HTML, CSS, JS, PHP y SQL para desarrollar una aplicación para una empresa teatral.

Desarrollamos una web con 3 roles de usuario: administrador, profesorado y alumnos. El administrador puede gestionar obras, pagos, mandar mensajes mediante un chat interno y manipular la gestión de los usuarios; el profesor puede gestionar sus obras como cambiar alumnos de clase, mandar mensajes y la creación de nuevos eventos; y el alumno disponia de un calendario interactivo para ver sus clases asignadas, ver el estado de sus pagos y mandar mensajes al profesorado.

El proyecto incluye un sistema de autenticación, gestión de usuarios y un panel de administración para el control de las obras y las entradas. Diseñamos una interfaz intuitiva y atractiva, utilizando Figma para el diseño y GitHub para el control de versiones, la base del proyecto se realizó en PHP y SQL para la gestión de datos, y se implementó un sistema de autenticación y diferentes roles para los usuarios.

El proyecto fue desarrollado en equipo, lo que nos permitió aprender a trabajar de manera colaborativa utilizando metodologías ágiles y herramientas de control de versiones como Git y GitHub. Aprendimos a gestionar el tiempo y los recursos de manera eficiente, y a comunicarnos de forma efectiva para lograr un objetivo común.

Proyecto Web GSFit

Desarrollo de una plataforma web para un gimnasio, donde los usuarios pueden registrarse y hacer cosas según su rol, crear rutinas de entrenamiento y gestionar su progreso con enlaces a vídeos externos que explican como se realizan los ejercicios.
Todo conectado a una base de datos externa para almacenar la información de los usuarios y sus rutinas.


En este proyecto, he utilizado HTML, CSS y JavaScript para crear una interfaz de usuario atractiva y funcional. La aplicación permite a los usuarios registrarse, iniciar sesión y gestionar sus rutinas de entrenamiento de manera sencilla e intuitiva.
Opté por hacer un header transparente para mejorar la experiencia del usuario y facilitar la navegación por la web y la experiencia del usuario.
La aplicación cuenta con un sistema de autenticación, un panel de usuario y una página para crear sus rutinas.
